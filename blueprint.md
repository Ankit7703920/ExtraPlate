**App Name**: ExtraPlate

## Core Features:

- User Authentication: Secure user login/registration using Firebase Authentication (phone/email).
- User Profile: Create and manage user profiles with name, photo, and optional college ID.
- Create Donation: Users can post food donations with details like title, description, plate count, veg/non-veg tag, photo upload, GPS location, expiry time, and visibility settings (public/college).
- Nearby Donations: Display a list and map view of nearby donations, sorted by distance and time. AI tool filters by tags like veg/non-veg to help users quickly find foods that meet their dietary requirements.
- Claim System: Users can claim donations, triggering notifications to the donor. Donations are marked as claimed/picked up.
- Bhandara Events: Organizers can post Bhandara (community meal) events with location, time, and details. Users can view nearby events.
- College Groups: Enable college-only groups for internal food sharing and exchange within the college community.

## Style Guidelines:

- Primary color: Use a warm and inviting orange (#FFA500) to convey community and nourishment.
- Background color: Light beige (#F5F5DC) to create a clean and welcoming feel.
- Accent color: Green (#90EE90) to highlight important actions and tags like 'veg'.
- Body and headline font: 'PT Sans', a humanist sans-serif, to create a modern yet approachable feel.
- Use simple, clean icons to represent donation types, filters, and actions.
- Employ a card-based layout for donations and events to ensure clarity and easy scanning.
- Implement subtle animations on claiming a donation to give positive feedback.
